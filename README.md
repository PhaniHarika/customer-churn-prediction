# 📊 Customer Churn Prediction

End-to-end ML pipeline to predict customer churn using XGBoost with SHAP explainability.

## 🔴 Live Demo
👉 [Click here](https://customer-churn-prediction-dulc8gskwxsa2bdbgrayvu.streamlit.app/)

## 🛠️ Tech Stack
PostgreSQL • XGBoost • SHAP • FastAPI • Streamlit • Python

## ✨ Features
- Predict churn probability for any customer
- SHAP explainability — see which features drive churn
- REST API endpoint with FastAPI
- Interactive Streamlit dashboard

## 🚀 Run Locally
```bash
pip install -r requirements.txt
uvicorn api:app --reload
streamlit run app.py
```
